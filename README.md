## 🚀 Get It Running  
Let’s fire this up – it’s stupid easy!  

### Step 1: Snag It  
Grab the goods from GitHub:  
```bash
git clone https://github.com/IshaanCyberTech/SubEnum.git  
cd SubEnum
python subenum.py
```
---

⚡ How to use It
Once you’re in, it’s all you!

⚡Quick Commands
Here’s your cheat sheet to dominate:

Command	What It Does
run	Start subdomain enumeration
config	Show current configuration
set <key> <value>	Set configuration (e.g., set mode scraping)
save	Save current configuration
status	Show current operation status
clear	Clear the terminal screen
help	Show the help message
exit	Exit the interactive mode
Example Attack:
Type run.
Throw in a target – example.com.
Pick your vibe – brute-force or scraping.
Watch subdomains stack up like trophies!
